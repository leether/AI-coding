# AI辅助编程实战指南 - 示例代码和资源

本目录包含书中提到的示例代码、配置文件和资源。

## 目录结构

```
examples/
├── podcast-app/          # 播客应用案例资源
│   ├── diagrams/         # 架构图
│   ├── prompts/          # 提示词模板
│   └── scripts/         # 实用脚本
├── templates/           # 文档模板
└── tools/              # 辅助工具
```

## 使用说明

### 提示词模板

所有提示词模板按章节组织在 `prompts/` 目录下，可以直接复制使用。

- 原始代码：展示重构前状态
- 重构代码：展示重构后状态
- 测试代码：完整测试用例
